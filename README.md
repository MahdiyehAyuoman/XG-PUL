
# XG-PUL

XG-PUL is a Positive-Unlabeled Learning framework for disease module identification based on human protein-protein interaction networks.

## Overview

This repository contains the implementation of the proposed XG-PUL method.

The method integrates node embedding, topological network features, and XGBoost-based PU learning for disease gene prioritization.

## Requirements

Python 3.x

Required packages are listed in requirements.txt.

## Data

The datasets used in this study include BioGRID and DisGeNET.
Due to data usage policies, the original datasets should be downloaded from their official sources.
